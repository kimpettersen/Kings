About kings cup

This is a drinking game that are supposed to create joy at a social gathering. It is NOT recommended to play this with alcohol!

This is how you set the game up

  1. Find a big cup and set it on the table
  2. Enter the amount of players
  3. Enter the name of the players, it helps if you do it in the order you sit around the table
  4. Assign rules to cards. BE CREATIVE
  5. Start playing

This is how you play

  1. Draw a card using space, enter or by clicking in the card area.
  2. If you get a king, you have to pure some of your drink into the cup. If you get the last king, you have to drink the cup
  3. If you get for example a 2, doesn't matter what kind of suite, you have to obey that rule until the game is over
  4. If you forget what rule is assigned to a card press "Remind me of this rule"
  5. HAVE FUN!
